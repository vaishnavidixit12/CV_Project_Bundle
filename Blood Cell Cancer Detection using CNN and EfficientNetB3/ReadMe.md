# Blood Cell Cancer Classification using CNN & EfficientNetB3

## Overview
Classifies blood cell images into cancerous and non-cancerous categories using CNN and EfficientNetB3 for improved accuracy.

## Dataset
Images sourced from [Kaggle](https://www.kaggle.com/) with folders for each blood cell type.
