# Fire Detection using YOLOv8

## Overview
Detects fire in images and videos using the YOLOv8 object detection model for real-time safety monitoring and environmental protection.

## Features
- Real-time fire detection with confidence scores  
- Easy YOLOv8 integration  
- Adaptable to various environments

Python 3.x, PyTorch, OpenCV, NumPy, Matplotlib
